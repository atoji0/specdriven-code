#### 写真・手書きキャンバス実装仕様

**使用ライブラリ**: `konva` + `vue-konva`

**実装概要**:

1. **カメラプレビュー**
   - `<video ref="videoEl" autoplay playsinline>` でカメラ映像を表示
   - `navigator.mediaDevices.getUserMedia({ video: { facingMode: "environment" } })` でリアカメラを起動
   - フラッシュトグルは `ImageCapture` API の `setOptions({ fillLightMode })` で実装（非対応時は無効化）

2. **撮影処理**
   - `<canvas ref="captureCanvas">` に video フレームを `drawImage` してキャプチャ
   - キャプチャ後、`<v-stage>` に背景画像レイヤー（`v-image`）として配置
   - video を非表示にし、キャンバスを表示に切り替える

3. **Konva 手書きキャンバス**
   - `<v-stage>` に2レイヤー構成:
     - `imageLayer`: 撮影画像を `v-image` で表示
     - `drawLayer`: 手書きストロークを `v-line` で描画
   - `@mousedown` / `@touchstart` → 描画開始（`isDrawing = true`）
   - `@mousemove` / `@touchmove` → 現在のストロークに点を追加
   - `@mouseup` / `@touchend` → 描画終了（`strokes.push(currentStroke)`）
   - **Undo**: `strokes.pop()`
   - **クリア**: `strokes = []`

4. **色・太さ選択**
   - 色: `el-select` — 選択肢: `[{value:"red",label:"赤"},{value:"blue",label:"青"},{value:"yellow",label:"黄色"},{value:"white",label:"白"}]`
   - 太さ: `el-select` — 選択肢: `[{value:2,label:"細"},{value:5,label:"中"},{value:10,label:"太"}]`

5. **保存時の画像合成**
   - `stage.toDataURL({ mimeType: "image/jpeg", quality: 0.85 })` で手書き込み済み画像を取得 → `form.annotatedImage` にセット
   - サムネイル生成: 別 `<canvas>` に 200×150 でリサイズ描画 → `toDataURL()` → `form.thumbnailImage` にセット

#### UI状態切替ルール

| 状態             | 対象項目・ボタン               | 挙動                                         |
|----------------|:-----------------------------:|---------------------------------------------|
| 新規モード       | 削除ボタン                     | 非表示（`v-if="mode === 'edit'"`）           |
| 更新モード       | 保存して次を撮影ボタン           | 非表示（`v-if="mode === 'new'"`）            |
| 写真未撮影       | 保存 / 保存して次を撮影         | `:disabled="!hasPhoto"`                     |
| 写真未撮影       | Undo / クリア                  | `:disabled="!hasPhoto"`                     |
| 車両コード未入力  | 車種名                         | 空文字列を表示                               |

#### ボタン動作

| ボタン             | 処理                                                                          |
|-------------------|-----------------------------------------------------------------------------|
| バーコードスキャン  | 別途 `BarcodeDetector` API 対応ブラウザで実装（未対応時は手入力のみ案内）      |
| 撮影               | `capturePhoto()` を呼び出し。カメラ→キャンバス切り替え                        |
| フラッシュ          | `flashEnabled` トグル。`ImageCapture.setOptions()` に反映                    |
| Undo              | `strokes.pop()`。`drawLayer` を再描画                                        |
| クリア             | `strokes = []`。`drawLayer` を再描画                                         |
| 保存               | バリデーション → `defectRecordApi.create/update()` → `emit("saved")` → close |
| 保存して次を撮影    | 保存後に同じ vehicleId を引き継いで `open()` を新規モードで再呼び出し          |
| 削除               | `confirmMessageBox` → `defectRecordApi.remove()` → `emit("saved")` → close  |
