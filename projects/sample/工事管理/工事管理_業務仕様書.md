﻿# 工事管理 業務仕様書

## 目的

本書は、工事管理業務の骨格（関係者・業務フロー・ユースケース）を整理し、関係者間の認識を揃えることを目的とする。

---

## 対象範囲

- プロジェクトの登録・管理
- プロジェクトに紐づく工事計画の一括登録・編集
- 工事計画のワークフロー申請・審査・承認

---

## 関係者（ロール）

| 関係者（実在）       | ロール（システム） | 主な責務                                                       |
| -------------------- | ----------------- | -------------------------------------------------------------- |
| 工事部 施工管理課    | 工事担当者 | プロジェクト・工事計画の管理およびワークフロー申請を行う       |
| 工事部 工事課長      | 審査者     | 申請された工事計画を審査する                                   |
| 工事部 部長（承認グループ固定） | 承認者 | 審査済みの工事計画を最終承認する                           |

---

## 業務の流れ

### マスタ管理

工事担当者がプロジェクトマスタを登録・管理する流れを表す。

#### 業務フロー

~~~plantuml
@startuml フロー1_マスタ管理
skinparam backgroundColor white
skinparam activityBorderColor #6c8ebf
skinparam activityBackgroundColor #ffffff

|工事担当者|
start

group 要選択 {
  :UC1:プロジェクト管理; <<#D6EAF8>>
}

stop
| |
@enduml
~~~

#### UC構成図

~~~plantuml
@startuml UC構成図_フロー1
left to right direction
actor 工事担当者

rectangle "プロジェクト管理" {
  (UC1: プロジェクト管理)
}

database プロジェクト
database 工事計画

工事担当者 --> (UC1: プロジェクト管理)

(UC1: プロジェクト管理) --> プロジェクト : <<create/update>>

プロジェクト ..> 工事計画 : <<ref>>
@enduml
~~~

### 工事計画登録・申請

工事担当者が工事内容を検討して計画を登録し、ワークフローで申請・承認を行う流れを表す。

#### 業務フロー

~~~plantuml
@startuml フロー2_工事計画登録・申請
skinparam backgroundColor white
skinparam activityBorderColor #6c8ebf
skinparam activityBackgroundColor #ffffff

|工事担当者|
start
:工事内容・実施時期を検討する;
:UC2:工事計画管理; <<#D6EAF8>>
:申請対象の工事計画を確認する;
:UC3:工事計画申請; <<#D6EAF8>>

|審査者 / 承認者|
:UC4:工事計画承認; <<#D6EAF8>>
if (承認か？) then (承認)
  stop
else (差し戻し)
  |工事担当者|
  :内容を修正して再申請する;
  stop
endif

@enduml
~~~

#### 状態遷移図

工事計画の申請から審査・承認の2段階承認を経て完了に至るステータス遷移を表す。

~~~plantuml
@startuml 状態遷移_工事計画
left to right direction

申請中 : 工事担当者
承認中 : 審査者 / 承認者
確認中 : 工事担当者
再申請中 : 工事担当者

[*] --> 申請中
申請中 --> 承認中 : 申請
承認中 --> 承認中 : 次ステップへ
承認中 --> 確認中 : 承認完了
確認中 --> [*] : 完了
承認中 --> 再申請中 : 差し戻し
再申請中 --> 承認中 : 再申請
再申請中 --> [*] : 取下げ
@enduml
~~~

**承認ステップ詳細**

| ステップ | 承認者 | 次ステップ | 差し戻し先 |
|:-------:|--------|:--------:|---------|
| 1 | 審査者 | 2 | 再申請中（工事担当者） |
| 2 | 承認者 | 承認完了 | 再申請中（工事担当者） |
