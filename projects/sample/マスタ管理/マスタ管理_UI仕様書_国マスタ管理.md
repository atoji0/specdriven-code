# UI仕様書 国マスタ管理

## 画面定義

- 画面ベース名：国マスタ管理
- 画面タイトル：国情報メンテナンス
- 画面種別：通常
- 入力方式：基本

---

## 画面概要

国情報の検索・一覧表示、および登録・更新・削除を行う画面。顧客情報に紐付いている国情報は削除できない。

### 画面ワイヤー

```plantuml
@startsalt
scale 1.2
{+
  { <b>国情報メンテナンス　　　　　　　　　　　　　　 | [ 新規作成 ] }
  --
  {
    国コード: | "　　　　"
    国名: | "　　　　　　　　　　　　　"
    通貨コード: | "　　　"
  }
  { [ 検索 ] | [ クリア ] }
  --
  {#
  国コード | 国名           | 通貨コード
  <U>JP   | 日本           | JPY
  <U>US   | アメリカ合衆国 | USD
  }
}
@endsalt
```

## 入力フォーム画面

### 画面レイアウト指示

特になし

### 画面ワイヤー

```plantuml
@startsalt
scale 1.2
{+
  { <b>国情報編集　　　　　　　　　　　　　　　　　　 }
  --
  {
    国コード: | "AA　"
    国名: | "　　　　　　　　　　　　　　"
    通貨コード: | "AAA"
  }
  --
  { [ 登録 ] | [ 更新 ] | [ 削除 ] }
}
@endsalt
```

### 項目定義（入力フォーム）

| 表示順 | 項目名     | UI部品       | 必須 | 入力制約/表示仕様                   |
|-------:|------------|--------------|:----:|-------------------------------------|
|      1 | 国コード   | テキスト入力 |  〇  | 2桁・半角英字（大文字）・重複不可   |
|      2 | 国名       | テキスト入力 |  〇  | 50桁以内                            |
|      3 | 通貨コード | テキスト入力 |  〇  | 3桁・半角英字（大文字）             |

### 項目間ルール（複合チェック）

特になし

### UI状態切替ルール

- 新規モード：国コードは入力可
- 更新モード：国コードは読み取り専用
- 顧客情報に紐付いている国情報は [削除] ボタンを非活性とする
