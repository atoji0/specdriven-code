# UI仕様書 不具合検索

## 画面定義

- 画面ベース名：不具合検索
- 画面タイトル：不具合検索
- 画面種別：参照

---

## 画面概要

過去の不具合日報を条件指定で検索・参照し、結果をExcelダウンロードできる。編集操作は行わない。

---

## 参照データ定義

参照_職場一覧：
- 取得元：職場マスタ
- 抽出条件：特になし
- 値：職場コード
- 表示：職場名

参照_車種一覧：
- 取得元：車種マスタ
- 抽出条件：特になし
- 値：車種コード
- 表示：車種名

参照_シフト区分一覧：
- 取得元：固定値

---

## 不具合検索画面

### 画面レイアウト指示

特になし

### 画面ワイヤー

~~~plantuml
@startsalt
scale 1.2
{+
  { <b>不具合検索　　　　　　　　　　　　　　　　　　 }
  --
  {
    職場: | ^職場を選択^
    車種: | ^車種を選択^
    日付（From）: | "YYYY-MM-DD"
    日付（To）: | "YYYY-MM-DD"
  }
  { [ 検索 ] | [ クリア ] | [ Excelダウンロード ] }
  --
  {#
  No | 日付       | 職場      | シフト | 不具合件数
  1  | 2026/03/01 | 〇〇職場 | 昼     | 3
  2  | 2026/03/01 | △△職場 | 夜     | 1
  }
}
@endsalt
~~~

### 項目定義（検索条件）

| 表示順 | 項目名       | UI部品         | 必須 | 入力制約/表示仕様              |
|-------:|--------------|----------------|:----:|-------------------------------|
|      1 | 職場         | プルダウン入力 | -    | 参照：参照_職場一覧           |
|      2 | 車種         | プルダウン入力 | -    | 参照：参照_車種一覧           |
|      3 | 日付（From） | 日付入力       | -    | -                             |
|      4 | 日付（To）   | 日付入力       | -    | -                             |

### 項目定義（一覧）

| 表示順 | 項目名     | UI部品       | 必須 | 入力制約/表示仕様              |
|-------:|------------|--------------|:----:|-------------------------------|
|      1 | No         | テキスト表示 | -    | -                             |
|      2 | 日付       | テキスト表示 | -    | YYYY/MM/DD 形式               |
|      3 | 職場       | テキスト表示 | -    | 参照：参照_職場一覧           |
|      4 | シフト     | テキスト表示 | -    | 参照：参照_シフト区分一覧     |
|      5 | 不具合件数 | テキスト表示 | -    | -                             |

### 出力定義（Excel）

- 出力単位：不具合 1件を 1行として出力する。
- 出力対象：検索条件で絞り込まれた全件（ページングなし）。

| 出力順 | 列名           | 出力内容                               |
|-------:|----------------|----------------------------------------|
|      1 | 日付           | 稼働日（YYYY/MM/DD 形式）              |
|      2 | 職場名         | 職場名                                 |
|      3 | シフト区分     | シフト区分                             |
|      4 | VINナンバー    | VINナンバー                            |
|      5 | 車種名         | 車種名                                 |
|      6 | 不具合部位     | 不具合部位                             |
|      7 | 不具合内容     | 不具合内容                             |
|      8 | 品質ランク     | 品質ランク                             |
|      9 | 備考           | 備考                                   |
|     10 | 記入者         | 記入者                                 |
|     11 | 登録日時       | 登録日時（YYYY/MM/DD HH:MM 形式）      |

### 検索仕様ルール

- 取得対象外条件：特になし
- ソート順：日付 降順、職場コード 昇順、シフト区分 昇順

### 項目間ルール（複合チェック）

- 日付（From）≦ 日付（To）であること

### UI状態切替ルール

特になし
