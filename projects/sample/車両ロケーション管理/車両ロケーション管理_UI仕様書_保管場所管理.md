# UI仕様書 保管場所管理

## 画面定義

- 画面ベース名：保管場所管理
- 画面タイトル：保管場所管理
- 画面種別：通常
- 入力方式：基本

---

## 画面概要

保管場所マスタを登録・管理する。削除は他データから参照されている場合は実行できず、その場合は有効フラグを無効にして利用停止とする。

## 入力フォーム画面

### 画面レイアウト指示

特になし

### 画面ワイヤー

```plantuml
@startsalt
scale 1.2
{+
  { <b>保管場所編集　　　　　　　　　　　　　　　　　 }
  --
  {
    保管場所ID: | "　　　"
    工場: | "　　　　　　　　　　"
    エリア: | "　　　　　　　　　　"
    階: | "999"
    スパン: | "　　　"
    通り: | "　　　"
    収容台数: | "9999"
    [X] 有効
  }
  --
  { [ 登録 ] | [ 更新 ] | [ 削除 ] }
}
@endsalt
```

### 項目定義（入力フォーム）

| 表示順 | 項目名     | UI部品           | 必須 | 入力制約/表示仕様                               |
|--------|------------|------------------|:----:|------------------------------------------------|
| 1      | 保管場所ID | テキスト入力     | 〇   | 最大4文字、半角英数字（大文字）、重複不可       |
| 2      | 工場       | テキスト入力     | 〇   | 最大100文字                                    |
| 3      | エリア     | テキスト入力     | 〇   | 最大100文字                                    |
| 4      | 階         | 数値入力         | -    | 0〜99の整数                                    |
| 5      | スパン     | テキスト入力     | -    | 最大20文字、半角英数字（大文字）                |
| 6      | 通り       | テキスト入力     | -    | 最大20文字、半角英数字（大文字）                |
| 7      | 収容台数   | 数値入力         | 〇   | 1〜1000の整数                                  |
| 8      | 有効       | チェックボックス | -    | -                                              |

### 項目間ルール（複合チェック）

特になし

### UI状態切替ルール

- 新規モード
  - 保管場所IDは入力可。
- 更新モード
  - 保管場所IDは編集不可（テキスト表示）。

---

## 操作

- [新規作成] ボタン押下
  - ②入力フォームを新規モードで表示する。
- 保管場所ID リンク押下
  - ②入力フォームを更新モードで表示する。
